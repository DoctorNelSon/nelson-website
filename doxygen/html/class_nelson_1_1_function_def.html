Detailed Description

Base class for the function types A FunctionDef class is a base class for the different types of function pointers used. There are three types of functions available:

  • M-functions - these are functions or scripts written by the user in the interpreted language.
  • Built-in functions - these are functions coded in C++ that implement functionality too difficult/impossible to do in the language itself. All of these functions have in common a name, a script classification (is it a script or not), a well defined number of input arguments, a well defined number of output arguments, and some means of being evaluated.
